Personal Projects

One of my personal projects, Summer Trends Analysis, involves analyzing keyword and designer mentions from a British Vogue article on top fashion designers. I utilized Python and Matplotlib to visualize the frequency of these mentions, which provided insights into the current fashion trends, showing who were the top popualar designers this season. This project not only honed my data analysis and visualization skills but also gave me a deeper understanding of trend analysis in the fashion industry.

Another significant project is NVDA Stock Analysis, where I was learning how to develop a machine learning model using Random Forest Regression. This heleped me analyze Nvidia (NVDA) stock prices based on historical data. I employed Plotly for interactive visualizations, enhancing the analysis of stock trends and patterns. This project allowed me to apply machine learning techniques to real-world financial data and improve my proficiency in model development and data visualization.
